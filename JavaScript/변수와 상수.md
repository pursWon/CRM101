변수와 상수
=====================

변수(variable)는 데이터를 저장할 때 쓰이는 ‘이름이 붙은 저장소’ 입니다.     

자바스크립트에선 let 키워드를 사용해 변수를 생성합니다.    

아래 문(statement)은 'message’라는 이름을 가진 변수를 생성(선언)합니다.    

</br>

```javascript

let message;

```

</br>

변수 선언과 값 할당을 한 줄에 작성하는 것이 가능합니다.     

```javascript

let message = 'Hello!'; // 변수를 정의하고 값을 할당합니다.

alert(message); // Hello!

```

</br>
