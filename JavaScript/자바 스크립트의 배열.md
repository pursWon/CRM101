자바 스크립트의 배열
==========================

### 대괄호를 사용해서 만드는 방법

```JavaScript
var arr = []; 

arr[0] = 'zero';
arr[1] = 'one';
arr[2] = 'tow';

for (var i = 0; i < arr.length; i++) {
    console.log(arr[i]);
}
```

</br>

```JavaScript
var arr = ['zero', 'one', 'tow']; 

for (var i = 0; i < arr.length; i++) {
    console.log(arr[i]);
}

```

</br>

```JavaScript
// 배열 생성 (배열 크기 지정)
// 쉼표 개수만큼 크기가 지정됨
var arr = [,,,]; 

for (var i = 0; i < arr.length; i++) {
    console.log(arr[i]);
}
        
// 값이 할당되지 않아서 undefined 3번 출력  

```
